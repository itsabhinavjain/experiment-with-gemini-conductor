# Specification: Track 03 - Next.js Web Frontend

## Goal
Develop a minimal, educational web interface for the Claude Agent SDK using Next.js.

## Requirements
- **Stack:**
  - Next.js (App Router)
  - React
  - TypeScript
  - Tailwind CSS (or minimal styling)
- **Features:**
  - Connect to `python-backend` (NOT Claude directly).
  - Simple chat UI with message history.
  - Streaming response support.
  - Display "Thinking Blocks" in a collapsible or clearly separated panel.
  - Educational UI layout: Minimalist and clear.
- **Folder Structure:**
  - `nextjs-frontend/` (MANDATORY NAME)
- **UI Layout:**
  - `[ Chat History ]`
  - `User message`
  - `Assistant response`
  - `Thinking panel` (collapsible)

## Design Goals
- Avoid heavy design frameworks.
- Keep it educational and easy to understand for beginners.
