# Specification: Track 04 - Educational Documentation

## Goal
Create a comprehensive documentation set for beginners that explains the architecture and implementation of the Claude Agent SDK reference project.

## Required Files
- **`architecture.md`**: System overview, data flow, agent lifecycle, ASCII diagram.
- **`backend-explained.md`**: SDK usage, sessions, streaming, thinking blocks.
- **`frontend-python.md`**: CLI design, streaming handling, rendering.
- **`frontend-nextjs.md`**: API integration, UI rendering, state handling.
- **`beginner-guide.md`**: Simple explanation of Agents, SDK, Thinking blocks, `uv`, and system connectivity.

## Requirements
- **Tone:** Technical Educator (Senior Engineer explaining to beginners).
- **Format:** Markdown.
- **Location:** `documentation/` folder in project root.

## Content Guidelines
- Use clear, simple language.
- Provide "Why" behind architectural decisions.
- Include code snippets where helpful.
- Ensure all diagrams are readable in plain text (ASCII).
